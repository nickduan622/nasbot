# NAS Project

NAS 自动化影视资源下载与整理系统，运行在绿联 NAS (Docker) 上。

## Workspace
- Path: `/home/ubuntu/nas`

## Tech Stack
- Runtime: Docker containers on UGREEN NAS (UGOS)
- Download client: qBittorrent
- Core stack: Prowlarr + Radarr + Sonarr + ChineseSubFinder
- Bot interface: Telegram Bot (or Overseerr + Requestrr)
